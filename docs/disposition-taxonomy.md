# Disposition Taxonomy — 18 Categories

| Code                  | Label                          | Description                              |
|---|---|---|
| CONVERTED             | Converted                      | Prospect agreed to buy                   |
| INTERESTED_FOLLOWUP   | Interested — Follow-up Booked  | Positive, next step scheduled            |
| INTERESTED_NO_NEXTSTEP| Interested — No Next Step      | Warm but no commitment made              |
| OBJECTION_PRICE       | Objection: Price               | Concerned about cost                     |
| OBJECTION_TIMING      | Objection: Timing              | Not the right time                       |
| OBJECTION_AUTHORITY   | Objection: Authority           | Needs to check with others               |
| OBJECTION_NEED        | Objection: Need                | Does not see the value                   |
| OBJECTION_COMPETITOR  | Objection: Competitor          | Evaluating a competing product           |
| NOT_INTERESTED        | Not Interested                 | Flat rejection                           |
| CALLBACK_REQUESTED    | Callback Requested             | Asked to be called back at a later time  |
| VOICEMAIL             | Voicemail                      | Left a voicemail message                 |
| NO_ANSWER             | No Answer                      | Nobody picked up                         |
| WRONG_NUMBER          | Wrong Number                   | Incorrect contact reached                |
| GATEKEEPER            | Gatekeeper                     | Did not reach the decision maker         |
| DNC                   | Do Not Call                    | Prospect requested no further contact    |
| PARTIAL_CALL          | Partial Call                   | Call dropped or cut short                |
| LANGUAGE_BARRIER      | Language Barrier               | Communication was not possible           |
| OTHER                 | Other                          | Does not fit any other category          |
